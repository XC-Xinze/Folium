# 卢曼实际怎么编号

数字化保存版本：[https://niklas-luhmann-archiv.de/](https://niklas-luhmann-archiv.de/)

观察：
- 大约 90,000 张卡，分两个主要 collection（早期 + 晚期）
- 编号深度可达 7-8 层
- 没有"清晰的总目录" —— 真·涌现结构
- 引用密集，单卡平均 3-5 个 [[link]]

跟我们软件的对应：
- 我们用 [[1a1b]] 这种连写表示，他用 `1a1/b` 加分隔符 —— 等价
- 他没有标签系统，纯靠 [[link]] + 一份 *Schlagwortregister*（关键词表）
- 我们的标签 = 他的关键词表 + 自动化

详见 [[1a1]] 命名实例的对照。
