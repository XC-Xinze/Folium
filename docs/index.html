1.5

Folium 1.5 is the first version shaped around the final interaction model: lighter main canvas, clearer workspace semantics, stronger visual identity, and fewer confusing filters.

From feature-heavy 1.0 to usable 1.5.

Folium 1.0 proved the core platform: local vaults, Electron packaging, workspaces, graph view, attachments, backups, and plugin groundwork. It was powerful, but some interactions were too eager.

  • 1.5 keeps box structure visible while stopping imported link cards from expanding endlessly.
  • Potential links are now an optional assistant layer, not a second structure system.
  • GraphView was reduced to the concepts users actually think with: Box, Link, and Tag.
  • Workspaces now better separate temporary reasoning from durable vault links.

Version history as product direction.

1.0

The first public build established the foundation: local Markdown vaults, Electron desktop packaging, card editing, workspaces, graph view, attachments, backups, and the initial plugin path.

  • Local-first vault with SQLite indexing.
  • Workspace canvases for temporary visual thinking.
  • Basic graph, tag, attachment, and export groundwork.
1.5

The 1.5 release simplifies the interaction model. It keeps Folium's exploratory spirit while removing recursive card flooding, unclear filters, and visual noise from the main canvas.

  • Cleaner box view with focused potential links.
  • Graph view grouped by top-level boxes with calmer focus behavior.
  • Workspace link labels, image export, first-run vault setup, and refined documentation.
