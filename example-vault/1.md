# 加密流量分类的核心问题

在 TLS 1.3 全面普及之后，传统基于深度包检测（DPI）的流量分类方法基本失效。研究重点已经从 payload 内容转向**侧信道特征**：包长度序列、包到达时间间隔、TLS 握手指纹等。

这构成了我们整个研究项目的起点。后续讨论展开为两条主线：

1. 高维特征空间下的机器学习方法（见 [[1a]]）
2. 时序信息的独立利用（见 [[1b]]）

整个研究框架受到 [[3b]] 中关于核方法跨领域应用的启发。
