Local-first linked cards

Think through cards, not folders.

Folium is a desktop knowledge workspace where cards, links, graph views, and temporary canvases help a thought stay alive long enough to become structure.

Pull cards into a workspace, annotate the relationships, and write real double-links back only when the connection earns its place in the vault.

A quieter tool for serious thinking.

Plain Markdown vaults

Your source of truth is ordinary Markdown. SQLite is only a fast local index.

Chain reading

Open a card and walk its relationships without losing the path that brought you there.

Workspace canvases

Use temporary cards, notes, colored relation labels, and apply-to-vault links when ready.

Graph focus

Explore structure visually while dimming unrelated links around the selected card.

Potential links

Surface latent relationships through local text analysis, without sending notes to a model.

Desktop safety

The packaged app uses a local backend, per-run API token, and guarded vault file routes.

Structure should emerge from use, not be imposed before the idea is ready.

Folium keeps the Obsidian virtues that matter: local files, extensibility, and an interface that can disappear when you are working. But it treats links as active routes through thought, not just metadata around notes.

Workspaces are the temporary layer: messy, provisional, visual. The vault is the durable layer: cleaner, calmer, and written back only when the relation is worth preserving.
