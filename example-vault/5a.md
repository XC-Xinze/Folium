# 不确定性采样

最朴素的查询策略：取分类器预测概率最分散的样本（最大熵）。

实现就一行：
```
score(x) = -sum_i p_i(x) * log p_i(x)
```

但这种 pointwise 方法在 batch 标注场景下会塌成"全选相似的难样本"。后来出现了 BatchBALD、Coreset 等方法考虑样本间的多样性。

实践中发现，结合委员会查询的混合策略更稳——单一 entropy 容易被维度灾难放大噪声。
