# 画布上每条边的来源

Canvas 显示的关系按"信号强度"分层：

| 层 | 颜色 | 行为 | 来源 |
|---|---|---|---|
| Tree | 灰实线 | 不可改 | Folgezettel 父子 |
| Tag | 绿实线 | **Trail 累加** | 共享 tag |
| Cross | 紫实线 | 单层，可 unlink | 手动 [[link]] |
| Potential | 灰虚线 | 单层，可 promote 成 cross | 文本相似 / 被 mention |

## Tag Trail 是什么

每次焦点切到一张**有 tag 的卡** → 它加进"tag 锚点集合"。所有锚点的 tag 邻居在画布上累加。box 切换才清空。

效果：你点 [[1a]] 看到 #zettelkasten 邻居 → 再点 [[2a]]（#tools）→ 上一批邻居**不消失**，新邻居叠加。探索过程被画布"记住"。

## Potential 单层

潜在关系是**信号**，不传染。点了 promote → 升级成紫色双链。
