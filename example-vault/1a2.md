# 插入新分支

Folgezettel 最反直觉的优势：**永远不需要重排**。

传统层级（folder / 一些笔记软件）的痛点：
- 想在 `chapter-3` 和 `chapter-4` 之间插一章 → 要么改后续编号，要么用 `chapter-3.5` 这种丑陋写法

Folgezettel 解法：
- 想在 `1a2` 和 `1a3` 之间补一个 → 不需要重排。直接在 `1a2` 下挂 `1a2a`、`1a2b` 拓展子分支
- 或者在 `1a3` 下挂 `1a3a` 反驳前面的观点

每张 id 是**永久地址**，外部 [[link]] 不会因为你重组结构而失效。

跟 [[1b]]（atomic notes）一起用威力最大。
