# 一个 drag handle 三件事

每张卡有两个拖动入口：

## 左上角的 luhmann id 徽章

最直观的"这张卡的身份"。拖它：
- → 另一张卡 = 写 `[[本卡]]` 进对方 body（建链）
- → sidebar 树节点 = reparent + 自动 Folgezettel 重编号
- → workspace = 加进 workspace

## 右下角的 DRAG 按钮

跟徽章功能一样，但更显眼，给不熟悉 id 拖拽的用户用。

## Sidebar Folgezettel 行

整行可拖：
- → canvas 上某张卡 = 建链
- → sidebar 另一节点 = reparent
- → workspace tab = 加进 workspace

## 关键设计

**一个对象，一种拖动语义**。drop target 决定操作类型。用户不需要记"哪种拖发起哪种动作"。
