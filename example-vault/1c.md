# 链接的三种本质强度

跟我们软件的视图层一一对应：

## 1. Tag 共现（绿线）—— 主题层

最本质。两张卡共享 #methodology → 它们在谈同类问题。这是**涌现的语义关系**，不需要你主动连。

## 2. Manual [[link]]（紫线）—— 论证层

你写 [[1a]] 时，你在主动断言："理解这张卡需要先理解 1a"。这是**意图明确的引用**。

## 3. Potential（灰虚线）—— 信号层

文本相似 / 一方提到了另一方但没 [[link]]。**只是提示**，不是关系。点 promote 按钮可以升级成紫线。

---

设计哲学：**视图把这三种强度可视化分开**，让你一眼分清"这张卡跟它周围卡的关系本质"。
