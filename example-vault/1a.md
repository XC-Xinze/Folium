# Folgezettel 编号系统

数字与字母**交替**编号 —— 这不是装饰，是结构语言：

| id | 含义 |
|---|---|
| `1` | 顶级盒子 |
| `1a` | 1 的第一个子卡 |
| `1a1` | 1a 的第一个子子卡 |
| `1a1b` | 1a1 的第二个子子子卡 |

## 为什么这样设计

- **id 本身就是路径**：看到 `2c3a` 立刻知道它在哪
- **可以无限延伸**：不像 folder 树，分支深度不受目录嵌套限制
- **可以无序生长**：今天写 `1a3`，明天插入 `1a4`，不影响旧卡

详见 [[1a1]] 看具体例子，[[1a2]] 看如何插入新分支。
