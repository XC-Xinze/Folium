# 原子判断的两个测试

**Test 1 — 引用测试**：你能想象未来某张卡引用这张卡时只引用它的一部分吗？如果能，那"一部分"应该是独立的卡。

**Test 2 — 标题测试**：能不能用一个 5-10 字的标题概括它？如果不行，混入了多个想法。

跟 [[2a]]（本软件的设计）相关：我们刻意没做 block-level reference（Logseq 的特色），就是**鼓励原子化**。想 reference 一段话？把那段话拆成独立的卡。
