# 委员会查询

Query by Committee：训 N 个子模型，对未标样本各自打分，看分歧（vote entropy / KL）最大的那批拿去标。

它隐式假设了：**子模型之间的多样性能折射出真实的不确定性**。如果所有子模型用同样的特征同样的初始化，分歧不过是噪声。

构造多样性的几种做法：
- bagging 样本扰动
- 不同特征子集（feature bagging）
- 不同架构

后两种特别适合加密流量这种特征异构的场景。和不确定性采样组合使用通常能再涨几个点。
