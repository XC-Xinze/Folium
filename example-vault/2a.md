# INDEX 不是手动选的

跟其他笔记软件的最大区别。

## 派生规则

> 一张卡有任何 Folgezettel 子卡 → 它自动是 INDEX。否则就是 ATOMIC。

跟 [[1]] 这张卡你正在看：它是 INDEX，因为有 [[1a]]、[[1b]]、[[1c]]。
[[1c]] 没有子卡 → 它就是 ATOMIC。
当你给 [[1c]] 加一个 [[1c1]]，[[1c]] 自动变 INDEX。

## 为什么这么设计

写卡的时候你**不知道**它会不会成为 INDEX。先写，后归类。这跟 [[1b]]（atomic notes）哲学一致。

如果让用户写卡时选 INDEX/ATOMIC：
- 选错了要改
- 决策成本侵占思考精力
- 跟"先写再结构化"的卡片笔记法精神矛盾

#tools 派生比手动好。
