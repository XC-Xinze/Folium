# Workspace —— 抽屉外的草稿台

vault 是**永久档案**，要谨慎结构化。但写笔记是流动的：你常常有"这几张卡放一起想想"的需求。

## Workspace 解决什么

- 把任意几张实体卡**临时拖到一起**，自由排版、画连线
- 加入 **temp 卡**（草稿）—— 没 luhmann id，只在 workspace 里
- 临时连线只在 workspace 里 —— 不污染 vault

## 提权流程

某张 temp 卡想正式纳入 vault：
1. 点卡上的 ↑ Promote 按钮
2. 软件根据它在 workspace 里的连接对象**自动建议** parent
   - 1 个连接 → 直接 next-available 子 id
   - 多个 → prompt 让你选哪个 parent
3. 提权时所有触及 temp 的 workspace 边自动物化成 [[link]]

跟 Logseq 的 daily journal 思路相通：**先放飞后归档**。
